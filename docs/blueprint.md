# **App Name**: DAGM Sponsorship Portal

## Core Features:

- Tiered Sponsorship Options: Allow users to select predefined sponsorship tiers with set amounts or enter a custom amount to support DAGM.
- QR Code Generation: Generate a unique QR code based on the selected or custom sponsorship amount for easy payment via QRIS.
- Transaction ID Display: Display a unique transaction ID associated with each sponsorship payment for tracking purposes.

## Style Guidelines:

- Primary color: Dark purple (#7231B5) to convey sophistication and support the other visual design choices in the prompt. (#7231B5)
- Background color: Very dark gray (#0A0A0B), nearly black, complementing the purple tone (#0A0A0B)
- Accent color: Fuchsia (#D926A9), for contrast to the purple primary color (#D926A9)
- Body and headline font: 'Plus Jakarta Sans', a sans-serif as specified in the user prompt. Note: currently only Google Fonts are supported.
- Use consistent and modern icons as inline SVGs, aligning with the design's tech-forward aesthetic.
- Implement a split-screen layout with key information on the left and interactive sponsorship options on the right.
- Incorporate subtle animations, such as fading effects, to enhance user engagement.
